# 需求摘要

    1.前置背景：使用MinerU将AFSIM使用手册拆分的63个pdf文档解析为md文档；
    2.需求概述：将63个md文档进行分块向gml-4.7-flash大模型提问，返回微调训练数据集；

# 需求描述

    1.配置gml大模型API Key；
    2.创建客户端连接，加上system内容：你是一个SFT微调的数据集生成器，该数据集主要用于微调大模型进行AFSIM想定脚本开发与知识问答，基于用户提问的AFSIM使用手册的切片内容进行多条数据集生成，你只需要输出数据集的json格式，风格为AIpaca，不要输出其他任何内容‘
    3.按照顺序读取某文件夹下的md文档，对md文档进行切片控制控制在2000-5000token内，如果该md末尾的切片不足2000则依旧作为一个切片；
    4.每轮对话的提问和大模型输出的数据集进行存储；
    5.提取关键对话内容：将对话交给AI让其对关键提问信息进行总结，同时将对话输出的数据集总结为类似“生成了AIpaca数据集，例如...”；
    6.滑动存储5轮历史对话的关键对话内容；
    7.将历史关键对话内容+system+切片作为提问。
